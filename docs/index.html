Features

Everything a real HGS does

Every symbol computed from live SimVar data, projected in real-time at sim framerate.

🎯
Conformal Runway Projection
8-corner perspective transform. The runway box is mathematically locked to the real pavement — not a fixed overlay.
✈️
Flight Path Vector
The FPV (bird) shows where the aircraft is actually going — not where the nose is pointing.
📡
ILS Confidence Rendering
5 render states: solid → dimmed → dashed → oscillating → hidden. Signal quality is always visible.
🛬
CAT III Approach
Decision height down to 0 ft. Flare director cue, rollout centerline guidance — full zero-visibility support.
💚
Phosphor Persistence
Accumulation buffer renderer simulates the afterglow of a real Collins HGS-4000 combiner glass.
🔧
One-Click Installer
Auto-detects MSFS Community folder. Creates timestamped backups. Self-heals if aircraft updates overwrite files.

Installation

Up and running in 4 steps

The installer handles everything automatically — no manual file editing required.

1

Download the installer

Download C_HUD_Install_v1.0.0.exe from the GitHub Releases page.

2

Run the installer

Double-click the EXE. Your MSFS Community folder is detected automatically — no setup needed.

3

Click 설치하기

The installer scans for compatible aircraft, creates backups, and patches the necessary files.

4

Launch MSFS and fly

Load a supported aircraft, activate the HUD power switch, and the overlay appears automatically.
